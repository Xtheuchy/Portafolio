Diseñé y desarrollé una página web estática para la venta y exhibición de ropa. Este proyecto implicó la creación de una interfaz visualmente atractiva y funcional, utilizando HTML y CSS para presentar productos de manera clara y profesional. Implementé características como una galería de imágenes de productos y un sistema de navegación intuitivo para mejorar la experiencia del usuario.

Web temática de Naruto

En este proyecto, desarrollé una página web dedicada al universo de Naruto. Me enfoqué en crear un diseño atractivo y bien organizado, facilitando la navegación y la visualización del contenido para fanáticos del anime. Implementé técnicas de diseño para destacar la información clave y asegurar una experiencia óptima en todos los dispositivos.
